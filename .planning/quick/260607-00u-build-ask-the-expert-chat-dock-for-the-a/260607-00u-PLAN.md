<objective>
Add an "Ask the expert" conversational surface to the analysis board so a user can
interrogate the EXISTING cached analysis (and get general expert advice) after an
analysis completes. The chat is the Apollo expert made interrogable — grounded
entirely on the already-persisted `analysis_results` row, so it costs ZERO new
engine spend. Answers stream token-by-token from Qwen on DashScope.

Purpose: Close the loop between the read-only InsightHero and the user's follow-up
questions ("why will this flop?", "how do I fix my hook?") without re-running the engine.

Output: `analysis_chats` migration + streaming chat API route (POST stream + GET
history) + dual-mode CommandBar with an upward-expanding DOM thread, seeded prompts,
scope chips, permalink replay, and a mobile full-height sheet.
</objective>
