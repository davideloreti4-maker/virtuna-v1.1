## Actionability Contract

     (a) NAMED ARCHETYPE + MECHANISM — the archetype slug (BOLD/GAP/CONTRARIAN/RESEARCH/NARRATIVE/QUESTION)

     — INTRA-BATCH DIVERSITY: when generating N hooks, each must use a DIFFERENT archetype slug.
