# Milestone: Linear Landing Clone

**Worktree identity.** Immutable after creation.

## What this milestone is

Ship a single-page landing site for Virtuna at top-tier production craft quality.

The bar for execution — typography rhythm, spacing rigor, scroll/motion polish, mobile responsiveness, micro-interactions — is benchmarked against best-in-class SaaS landing pages (Linear, Stripe, Vercel, Raycast). Linear's landing is the primary craft reference because the user identified it as the closest stylistic match for Virtuna's intended brand direction.

## Craft reference vs. content boundary

**Used as quality reference:**
- General typographic scale and rhythm (display weight, tracking, line-height patterns)
- Section pacing and scroll choreography common to top-tier SaaS landings
- Spacing and grid discipline
- Motion smoothness and restraint
- Mobile responsive behavior

**Never reproduced from Linear (or any reference):**
- Literal copy, headlines, or phrasing
- Specific illustrations, graphics, animations, or creative compositions
- Brand assets (logos, color palette, iconography)
- Distinctive section concepts unique to Linear's product narrative

Every word and every creative asset on Virtuna's landing page is **original to Virtuna**. Section composition is derived from Virtuna's product story (prediction, competitor intelligence, brand deals, behavioral science moat), not lifted from any reference site.

## Brand color

Coral `#FF7F50` stays. This is Virtuna's locked brand identity; we do not adopt any reference site's palette.

## Build location

This worktree at `~/virtuna-landing-linear-clone` on `milestone/landing-linear-clone`. The existing `~/virtuna-v1.1` main worktree stays clean.

## Visual fidelity gate

Per-phase verification: Playwright snapshots compare Virtuna's implementation against the craft quality bar. "Done" requires sign-off that the section reaches production polish — not pixel-match to any reference site.
