**Step 2 — Run baseline on 500-video stratified subsample:**

Substitute FULL_VERSION with the actual `full.YYYY-MM-DD` corpus_version from Plan 01-06.

```bash
npx tsx scripts/eval.ts \
  --corpus-version FULL_VERSION \
  --subsample 500 \
  --stratified \
  --seed 42 \
  --label v2.1-baseline \
  --baseline \
  --max-total-cost-cents 1500 \
  --output ".planning/benchmarks/$(date +%Y-%m-%d)-v2.1-baseline.json" 2>&1 | tee /tmp/baseline-eval.log
```
