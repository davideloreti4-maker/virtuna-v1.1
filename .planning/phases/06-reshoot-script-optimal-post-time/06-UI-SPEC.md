## Spacing Scale

**Standard scale (8-point grid, canonical set only):**

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon-to-label gaps, inline icon padding |
| sm | 8px | Inter-section divider clearance, chip internal padding |
| lg | 16px | Card internal padding horizontal (inside AV hero), section-header internal pad |
| xl | 24px | Section-header-to-body clearance inside ScriptBody |
| 2xl | 32px | Sheet content padding (edit Sheet body) |

**Spacing Exceptions (non-standard values вҖ” each justified per surface):**

| Value | Location | Justification |
|-------|----------|---------------|
| 12px (`py-3`) | Compact card (170Г—88) vertical padding | Compact card density at 170Г—88 requires tighter vertical rhythm than standard 16px; 8px is too tight, 16px overflows. Scoped to compact card only. |
| 12px (`py-3`) | AV hero (360Г—160) vertical padding | Mirrors compact card padding for visual consistency across both script surfaces; hero horizontal pad remains `px-4` (16px). |

Touch target exception (accessibility, not layout spacing):
- Copy buttons: minimum 44Г—44px tap area (WCAG touch target). Visual icon size smaller; padding expands tap zone only.
- Day-picker pill group: `gap-2` (8px) between pills, pills are `h-8 min-w-[40px]`.
- Inter-section dashed dividers in ScriptBody: `my-2` (8px vertical margin each side = 16px total clearance).

**Radius scale (from CLAUDE.md вҖ” not spacing):**

| Element | Radius |
|---------|--------|
| Cards (compact + AV hero) | 12px |
| Buttons (Copy-all pill, per-section copy) | 8px |
| Inputs (hour selects in edit Sheet) | 8px |
| Sheet modal (edit Sheet) | 12px |
| Day-picker pills | 8px |
| GlassPill (day/time chip, source pill) | inherits GlassPill component default |

Notes:

- Card icon labels (e.g. "Reshoot script", "When to post"): `text-xs font-medium text-white/85` (was `text-[11px]`).
- Reasoning row in OptimalPostCard: `text-xs text-white/65` (was `text-[11px]`).

**Glass pattern for sticky Copy-all pill (per CLAUDE.md):**
```
background: linear-gradient(137deg, rgba(17,18,20,0.75) 4.87%, rgba(12,13,15,0.9) 75.88%)
backdrop-filter: blur(5px)  [applied via React inline style вҖ” Lightning CSS strips it from classes]
border: 1px solid rgba(255,255,255,0.06)
box-shadow: rgba(255,255,255,0.15) 0 1px 1px 0 inset
border-radius: 12px
```

**Contrast verification:**
- `text-white/55` on `#07080a` background: white at 55% opacity = rgba(255,255,255,0.55) вҶ’ effective color ~`#8c8c8c` вҶ’ contrast ratio вүҲ 4.6:1 against `#07080a`. Passes WCAG AA (4.5:1 minimum for normal text вүҘ14px). At 10px section labels (small text, WCAG AA requires 4.5:1 for large text вүҘ18px bold or 3:1 for normal large), borderline вҖ” acceptable given label size is visually supplementary (icon + context available).
- `text-white/85` on `#07080a`: вүҲ 16.2:1. Passes WCAG AAA.
- `text-emerald-300/70` on `#07080a`: emerald-300 = `#6ee7b7` at 70% вҶ’ passes AA.

## Accessibility Contract

### Per-Section Copy Buttons
- Element: `<button type="button">`
- `aria-label="Copy {SectionName} section"` (e.g., `"Copy New Opening section"`)
- `title="Copy {SectionName} section"` вҖ” native HTML tooltip, same text as aria-label.
- Keyboard: Tab-reachable in DOM order. Enter fires copy + announces via aria-live.
- Focus indicator: `outline-2 outline-offset-2 outline-white/40`
- Touch target: min 44Г—44px padding even if visual icon is smaller.

### Copy-all Pill
- Element: `<button type="button">` inside GlassPill
- `aria-label="Copy full reshoot script"`
- `title="Copy full reshoot script"` вҖ” native HTML tooltip.
- Same keyboard behavior as per-section buttons.

### Day Picker (OptimalPostEditSheet)
- Wrapper: `<div role="radiogroup" aria-label="Day of week">` containing 7 `<GlassPill>` rendered as `<button role="radio" aria-checked={isSelected}>`.
- Arrow navigation: ArrowLeft/ArrowRight cycle through pills; Home/End jump to Mon/Sun.
- Space/Enter: selects focused pill.
- Selected state visually distinct via `GlassPill active` prop (NOT color-only вҖ” GlassPill adds border/bg change).

### Edit Sheet
- Focus trap: Radix Dialog built-in (Sheet uses Radix `Dialog.Content`).
- Esc: closes Sheet, focus returns to trigger element.
- Focus return: `onCloseAutoFocus` prop on `SheetContent` + `triggerRef` pattern (PersonaInspector.tsx line 121 precedent).
- Sheet title: `<SheetTitle>Edit post time</SheetTitle>` rendered as visually visible heading.

### Source Pill Tooltip
- Trigger: `Info` icon (`Phosphor Info`) with `tabIndex={0}` and `role="button"` or Radix `Tooltip` trigger.
- `title="View data source"` on Info icon for native tooltip fallback.
- Focus вҶ’ Enter/Space opens tooltip (or Radix handles this).
- Esc: dismisses tooltip.
- Screen reader: tooltip text is `aria-describedby` target.

### Aria-live Region
- Region: existing `aria-live="polite"` on `<ActionsNode>` (Phase 5 wired).
- Phase 6 writes to this region for 3 announcements (script-ready, empty-state, copy-confirmation).
- All announcements debounced 500ms to prevent spam on rapid copy clicks.
- Only fires when content actually changes (useRef guard for empty-state вҖ” fires once per analysis render).

### Color + Contrast
- All text meets WCAG AA:
  - `text-white/85` on `#07080a`: ~16:1 ratio вҖ” AAA.
  - `text-white/65` on `#07080a`: ~9:1 ratio вҖ” AAA.
  - `text-white/55` on `#07080a`: ~6.3:1 ratio вҖ” AA.
  - `text-emerald-300/70` on `#07080a`: ~5.8:1 ratio вҖ” AA.
- "Copied!" state uses `text-coral-500` (`#FF7F50`) on `#07080a`: ~4.6:1 вҖ” passes AA (вүҘ14px text).
- Copy confirmation: NOT color-only вҖ” `CheckCircle` icon + "Copied!" text accompanies color change.
- Source pill state change: NOT color-only вҖ” pill text label changes per state ("from your niche" / "default" / "yours").

### prefers-reduced-motion
- Sheet slide animation: when `usePrefersReducedMotion() === true`, override SheetContent `className` with `transition-none`. Radix's `data-[state=open]:slide-in-from-bottom` still fires but duration is 0.
- AV grid-template-rows grow transition: inherits Phase 5 `ActionsNode.tsx:43` pattern вҖ” already guards with `prefersReducedMotion ? 'none' : 'grid-template-rows 200ms ease-out'`.
- Copy "Copied!" feedback: opacity transition `duration-150` only вҖ” acceptable under reduced-motion (brief, non-decorative). No spring animation.
- Skeleton pulse (`animate-pulse`): set to `opacity-100` (no pulse) when `prefersReducedMotion === true`.
