    Implement `export function rankWithAudienceFit(ranked: RankedOutlier[], audience: Audience, serendipity: number): FitRankedOutlier[]`:

    - Else: for each tile compute `nm = nicheMatch`, `cf = calibrationFit`, `nicheWeight = 1 - serendipity` (clamp serendipity to 0..1 first), `fitScore = nicheWeight * nm + (1 - nicheWeight) * cf`. "Fair" : "Weak"` (use the EXACT thresholds 0.66 / 0.4 and the EXACT enum words `"Strong"|"Fair"|"Weak"` — Pitfall 4: a typo like "Medium" breaks the schema). Re-rank by a blended key `t.rankKey * (1 + 0.5 * fitScore)` (measured signal stays primary — α=0.5), sort desc. Return each tile spread with `fit: { level }`.

Add a file header comment stating: pure deterministic math, NO SIM call, NO fabricated reaction (D-02/D-03), lives at the runner/route layer NOT inside any engine scoring function (Pitfall 6, ENGINE_VERSION stays 3.19.0).
