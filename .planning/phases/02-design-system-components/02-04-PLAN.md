  <action>
    Create a FadeIn animation wrapper using Motion:

    **Requirements:**
    - Add "use client" directive (Motion requires client component)
    - Import motion from "motion/react"
    - Accept children, delay, duration, className props

    **Props interface:**
    ```typescript
    interface FadeInProps {
      children: React.ReactNode
      delay?: number
      duration?: number
      className?: string
    }
    ```

    **Default values:**
    - delay: 0
    - duration: 0.5

    **Animation:**
    - initial: { opacity: 0, y: 20 } (start invisible, slightly below)
    - animate: { opacity: 1, y: 0 } (fade in and move to position)
    - transition: { duration, delay, ease: [0.4, 0, 0.2, 1] }

    The ease curve [0.4, 0, 0.2, 1] is "ease-out" - starts fast, ends slow.

    Use motion.div as the wrapper element.
  </action>
