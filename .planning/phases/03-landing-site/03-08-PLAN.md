<task id="3">
<title>Add scroll-triggered animations</title>
<description>
Implement scroll-triggered entrance animations for all sections:

**Animation Pattern:**
Each section should fade in when scrolling into view.

**Using motion (v12):**
```tsx
import { motion, useInView } from "motion/react";
import { useRef } from "react";

function Section({ children }) {
  const ref = useRef(null);
  const isInView = useInView(ref, { once: true, margin: "-100px" });

  return (
    <motion.section
      ref={ref}
      initial={{ opacity: 0, y: 20 }}
      animate={isInView ? { opacity: 1, y: 0 } : {}}
      transition={{ duration: 0.6, ease: [0.215, 0.61, 0.355, 1] }}
    >
      {children}
    </motion.section>
  );
}
```

**Sections to animate:**
- BackersSection: Logos stagger in
- FeaturesSection: Header then cards stagger
- StatsSection: Left and right columns
- CaseStudySection: Card and quote
- PartnershipSection: Quote and card
- FAQSection: Header then accordion items

**Reduced Motion:**
- Check `useReducedMotion()` hook
- Disable animations if user prefers reduced motion

**Easing:** Use ease-out-cubic from design tokens
</description>
</task>
