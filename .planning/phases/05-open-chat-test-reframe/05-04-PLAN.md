<objective>
Add the conversation-layer affordances that make a skill run feel like a conversation: (1) **real pipeline-stage progress over SSE** (STUDIO-01 / D-02) — named stage events flipped to Perplexity-style checkmarks (Generating → Self-judge → Simulating your audience (SIM-1 Flash) → Ranking), no fake timers; (2) a short **model-authored follow-up chat turn** (STUDIO-02 / D-03) appended after the cards, referencing this run; (3) the **generic chain-handoff contract** (STUDIO-03 / D-09) so every skill declares its "→ next skill" CTAs + anchor-carry shape in one place — Script/Remix (P6) plug in with zero new plumbing; and (4) a **skill-run error state with tap-to-retry** (W2) rendering the UI-SPEC copy when a run drops out.

Purpose: STUDIO-01/02/03 + D-02/D-03/D-09 — the satisfying real-progress texture, the "it's a conversation, not just cards" payoff, the plumbing Phase 6 plugs into, and an honest, retry-able failure surface.
Output: `chain-handoff.ts` contract, named `stage` SSE events on the ideas + hooks routes, a `ProgressChecklist` transient UI, a model-authored follow-up turn persisted per run, and a skill-run error/retry surface in the thread views.
</objective>
