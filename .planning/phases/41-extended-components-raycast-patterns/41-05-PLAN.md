**CategoryTabs (`category-tabs.tsx`):**

A composition of the Tabs component (from 41-02) styled specifically for category navigation like Raycast's store/extensions page.

Interface:
```typescript
interface CategoryTab {
  value: string;
  label: string;
  icon?: React.ReactNode;
  count?: number;
}

interface CategoryTabsProps {
  categories: CategoryTab[];
  defaultValue?: string;
  value?: string;
  onValueChange?: (value: string) => void;
  children: React.ReactNode;
  className?: string;
}
```

Implementation:
- Import and compose `Tabs`, `TabsList`, `TabsTrigger`, `TabsContent` from `./tabs`
- TabsList wrapper: `overflow-x-auto scrollbar-none flex-nowrap` for horizontal scroll on mobile
- Each TabsTrigger renders: `{icon && <span className="mr-1.5">{icon}</span>}{label}{count !== undefined && <span className="ml-1.5 text-xs text-foreground-tertiary">({count})</span>}`
- TabsContent mapped to children using `React.Children` or requiring children as a render function

Simple composition approach: CategoryTabs renders the Tabs + TabsList with styled triggers. User provides TabsContent children matching category values.

```tsx
// Usage:
<CategoryTabs categories={[
  { value: "all", label: "All", count: 42 },
  { value: "productivity", label: "Productivity", icon: <Lightning />, count: 12 },
]}>
  <TabsContent value="all">...</TabsContent>
  <TabsContent value="productivity">...</TabsContent>
</CategoryTabs>
```
