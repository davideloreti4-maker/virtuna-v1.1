    Use Playwright MCP tools to extract design tokens from raycast.com homepage:

    1. Use `mcp__playwright__browser_navigate` to go to https://raycast.com
    2. Use `mcp__playwright__browser_snapshot` to capture the page state
    3. Use `mcp__playwright__browser_evaluate` to extract computed styles from key elements:

    Elements to inspect (use document.querySelector + getComputedStyle):
    - body: background-color, color, font-family
    - nav/header: background, blur values, border
    - h1 (hero title): font-size, font-weight, line-height, letter-spacing, color
    - hero subtitle text: font-size, color, line-height
    - primary button (CTA): background, color, padding, border-radius, box-shadow
    - secondary/ghost button: background, border, color
    - cards (if visible): background, border, border-radius, box-shadow

    JavaScript to run in browser_evaluate:
    ```javascript
    const getStyles = (selector, props) => {
      const el = document.querySelector(selector);
      if (!el) return { selector, found: false };
      const styles = window.getComputedStyle(el);
      const result = { selector, found: true };
      props.forEach(p => result[p] = styles.getPropertyValue(p));
      return result;
    };

    return {
      body: getStyles('body', ['background-color', 'color', 'font-family']),
      nav: getStyles('nav, header', ['background-color', 'backdrop-filter', '-webkit-backdrop-filter', 'border-bottom']),
      heroTitle: getStyles('h1', ['font-size', 'font-weight', 'line-height', 'letter-spacing', 'color']),
      heroSubtitle: getStyles('h1 + p, .hero p', ['font-size', 'color', 'line-height']),
      primaryBtn: getStyles('a[href*="download"], button.primary, [class*="primary"]', ['background-color', 'color', 'padding', 'border-radius', 'box-shadow']),
      card: getStyles('[class*="card"], article', ['background-color', 'border', 'border-radius', 'box-shadow'])
    };
    ```

    5. Use `mcp__playwright__browser_take_screenshot` to capture a screenshot for verification.
