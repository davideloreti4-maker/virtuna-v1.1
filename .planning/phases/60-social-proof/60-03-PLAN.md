# Plan 60-03: Logo Cloud Section

**Requirement:** SOCL-03 (Logo cloud marquee/grid)

## What

Create a `LogoCloudSection` displaying partner/integration logos in a continuous horizontal marquee animation, reinforcing ecosystem credibility. Uses CSS animation for the infinite scroll effect (no JS needed).

## Files

- CREATE `src/components/landing/logo-cloud-section.tsx`
- EDIT `src/components/landing/index.ts` (add export)

## Implementation

```tsx
// Server component (no "use client" needed — CSS-only animation)
// Uses SectionWrapper for consistent spacing
// Header: "Trusted by leading teams" or "Integrates with your stack"
// Logos: SVG placeholder logos (text-based for now, can swap for real logos later)
// Marquee: CSS @keyframes infinite scroll, duplicated logo row for seamless loop
// Gradient fade edges: left/right fade masks for clean entry/exit

```

### CSS Marquee Pattern

```tsx
// Outer container: overflow-hidden with gradient masks on left/right edges
// Inner track: flex row with gap, CSS animation translateX(-50%) over duplicated set
// Two copies of logos for seamless infinite loop
// prefers-reduced-motion: pause the animation
// Hover: pause animation (optional, good UX)

// CSS keyframes in component style or globals.css:
// @keyframes marquee { from { transform: translateX(0); } to { transform: translateX(-50%); } }
```

## Design

- Section label: `text-sm text-foreground-muted uppercase tracking-wider` centered
- Logos: `text-foreground-muted/50` (dimmed, not competing with content), `text-lg font-medium`
- Logo items have subtle border: `border border-white/[0.06] rounded-md px-6 py-3`
- Gradient fade masks on left/right edges (from bg-background to transparent)
- Animation duration: ~30s for smooth, slow scroll
- Gap between items: `gap-8`
- Respects prefers-reduced-motion: shows static grid instead of marquee

- prefers-reduced-motion shows static grid layout
