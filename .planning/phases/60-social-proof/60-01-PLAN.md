# Plan 60-01: Animated Stat Counters Section

**Requirement:** SOCL-01 (Animated stats/metrics counters)

## What

Create a `SocialProofStats` section with animated counters that count up from 0 to their target value when scrolled into view. Displays 3-4 key metrics (e.g., "500K+ videos analyzed", "10M+ data points", "50K+ trends tracked") conveying platform scale.

## Files

- CREATE `src/components/landing/social-proof-stats.tsx`
- CREATE `src/hooks/useCountUp.ts` (reusable count-up hook)
- EDIT `src/components/landing/index.ts` (add export)

## Implementation

### useCountUp Hook

```tsx
// Custom hook using motion/react useSpring for smooth count-up
// Triggers on IntersectionObserver entry (once: true)
// Supports formatted output (e.g., "500K+", "10M+")
// Respects prefers-reduced-motion (shows final value immediately)
interface UseCountUpOptions {
  end: number;
  duration?: number; // ms, default 2000
  suffix?: string;   // e.g., "K+", "M+", "%"
  prefix?: string;   // e.g., "$"
}
```

### SocialProofStats Component

```tsx
// Server component wrapper with client-side animated counter leaves
// Uses SectionWrapper for consistent spacing
// Grid: 3-4 stats in a row, responsive to 2-col on tablet, stack on mobile
// Each stat: large number (font-display, text-4xl+), label below (text-foreground-secondary)
// Subtle top/bottom border divider (border-white/[0.06])
// FadeInUp on each stat with stagger
```

### Stats Data

```tsx
const stats = [
  { value: 500, suffix: "K+", label: "Videos Analyzed" },
  { value: 10, suffix: "M+", label: "Data Points Collected" },

```
