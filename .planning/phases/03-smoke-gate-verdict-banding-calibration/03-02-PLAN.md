Calibrate the verdict banding (GATE-02 code change): add a symmetric dead-band buffer
around the VERDICT_BANDS thresholds (70 and 40) so near-boundary scores read "Mixed
signals" as a first-class verdict, with the buffer half-width provably wider than the
variance measured in Plan 01.

survives engine noise (kills "confident lie -> trust dies"). Because the determinism gate

"Mixed signals" would collapse to the antiViralityGated-only path (D-03).

additive cases in verdict.test.ts. NO new band row. NO engine edits. NO board-copy edits.
