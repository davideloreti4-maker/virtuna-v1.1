## v2.0 Requirements

Requirements for design system foundation. Each maps to roadmap phases.

### Extraction (EXT)

- [x] **EXT-01**: All Raycast color values extracted from raycast.com (backgrounds, text, accents, states)
- [x] **EXT-02**: All Raycast typography values extracted (fonts, sizes, weights, line heights, letter spacing)
- [x] **EXT-03**: All Raycast spacing values extracted (padding, margins, gaps)
- [x] **EXT-04**: All Raycast shadow values extracted (elevation levels, glows)
- [x] **EXT-05**: All Raycast border values extracted (radius scale, widths)
- [x] **EXT-06**: All Raycast animation values extracted (durations, easing curves)
- [x] **EXT-07**: All Raycast gradient definitions extracted
- [x] **EXT-08**: Extraction covers all interactive states (hover, focus, active, disabled)
- [x] **EXT-09**: Extraction documented with source references
- [x] **EXT-10**: Extract from all key pages: homepage, pricing, about, extensions, API docs
- [x] **EXT-11**: Exact font family identified (Inter vs custom) with fallback stack
- [x] **EXT-12**: Responsive breakpoints extracted (mobile, tablet, desktop thresholds)
- [x] **EXT-13**: Transition durations extracted per interaction type (hover, click, appear, exit)

### Tokens -- Colors (COL)

- [x] **COL-01**: Coral scale generated (100-900) with perceptually uniform progression
- [x] **COL-02**: Coral accessibility verified -- all text/background combinations pass WCAG AA (4.5:1)
- [x] **COL-03**: Darkened coral variant created for text on light backgrounds
- [x] **COL-04**: Primitive color tokens defined (raw values, not used directly)
- [x] **COL-05**: Semantic color tokens defined (bg, fg, accent, error, warning, success, info)
- [x] **COL-06**: Border color tokens defined
- [x] **COL-07**: State color tokens defined (hover, active, disabled variants)
- [x] **COL-08**: Color tokens match Raycast exactly (except coral replacing brand color)

- [x] **TYP-04**: Line height tokens defined per size
