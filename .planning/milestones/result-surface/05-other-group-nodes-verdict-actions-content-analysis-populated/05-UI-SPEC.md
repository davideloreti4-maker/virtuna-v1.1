## Spacing Scale

Declared values (multiples of 4):

| Token | Value | Usage in Phase 5 |
|-------|-------|------------------|
| xs | 4px | Icon-to-label gap, GlassPill internal padding, hook decomp bar row gap (`gap-1`) |
| sm | 8px | Frame internal padding, card gap |
| md | 16px | Frame body padding (8px each side = 16px total), section gap |
| lg | 24px | Collapsible body padding, inspector content spacing |
| xl | 32px | Headline strip + bar stack + chip row stacking budget |
| 2xl | 48px | — |
| 3xl | 64px | — |

**Exceptions:**

- Verdict frame: 8px padding all around (not 16px) — tight frame bounds (360×280)
- Actions frame: 8px frame padding; cards use 12px internal padding (`p-3`)
- Content-Analysis frame: 8px padding all around with 16px gap between Hook decomp and Emotion arc nodes
- SimilarVideoCardCompact: 8px internal padding (`p-2`), 56×56 thumb fixed size
- Anti-virality header band: 40px fixed height, 8px horizontal padding
- Collapsible summary row: 8px padding (`p-2`), not 16px
- `<details>` chevron icon: 12px size
- Hook decomp bar label width: 120px fixed (not a spacing token — layout constant)
- Touch targets on collapsible summaries: minimum 40px height (iOS tap target minimum)

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `#07080a` (`--color-background`, `--color-gray-950`) | Board canvas background, node body base |
| Secondary (30%) | `#18191a` (`--color-surface`, `--color-gray-900`) | Frame surfaces, card backgrounds, inspector sheet bg, collapsible open state (`open:bg-white/[0.04]`) |
| Accent (10%) | `oklch(0.72 0.16 40)` = `#FF7F50` (`--color-accent`, `--color-coral-500`) | Reserved list below |
| Warning / Anti-virality | `oklch(0.75 0.15 85)` (`--color-warning`) | Anti-virality header band ONLY |

**Accent (`--color-accent` / coral #FF7F50) reserved for:**

1. Verdict percentile number when `overall_score >= 70`
2. Anti-virality header gradient start (`coral → var(--color-warning)` horizontal, not a full glow)
3. Hook decomp weakest-modality bar background tint (`bg-accent/8` = 8% opacity)
4. Hook decomp GlassProgress bar fill (coral fill on dark track)
5. Emotion arc area chart: stroke line + gradient top stop (60% opacity) + peak dot (6px, filled coral)
6. Comparison chart: "current analysis" bar fill in both BarChart instances
7. Segment-pill timestamps in TopFixesList (subtle coral underline, `color: var(--color-accent)`)
8. Anti-virality "Post anyway →" link text

**Non-accent states:**

- Confidence chip: `white/60` text — contextual, not celebratory. No coral.
- History chart baseline bars: `rgba(255,255,255,0.30)` (white/30)
- Niche cohort p75 bar: `rgba(255,255,255,0.15)` (white/15)
- Emotion arc valley dots: `rgba(132,133,134,0.60)` (muted gray at 60%)
- Placeholder cards: `white/40` text, no accent
- Collapsible summary text: `white/95` foreground

**No red. No green.** Anti-virality treatment uses orange (`--color-warning`), not red. Score quality gradient uses coral-intensity on dark, not green-to-red. Per Phase 4 D-15.

**Borders:** Universal 6% (`rgba(255,255,255,0.06)`, `border-white/[0.06]`). Hover 10% (`border-white/[0.1]`). Placeholder cards use dashed border same 6% opacity. Per Raycast design language.

## Interaction States

| Node | State | Visual Treatment |
|------|-------|-----------------|
| Verdict | Streaming | `--` placeholder, confidence "Calculating…", collapsibles disabled |
| Verdict | Complete | Full content rendered |
| Verdict | Anti-virality | Orange header shown, collapsible pre-targeted to "Counterfactual considered" |
| Actions | Streaming | Placeholder cards static, SimilarVideos shows 5 skeleton rows |
| Actions | Complete | SimilarVideos populated from `result.retrieval_evidence` |
| Actions | Anti-virality | Frame grows to 360×360, ReshootHeroSlot spans full top half |
| Hook decomp | Streaming | 4 bars at 0% + shimmer (suppressed in reduced-motion) |
| Hook decomp | Complete | Bars filled to actual values |
| Hook decomp | Empty | 4 bars at 0%, caption rendered |
| Emotion arc | Streaming | Flat gray baseline |

| Emotion arc | Empty | Flat gray baseline + caption |

**Reduced motion (`prefers-reduced-motion: reduce`):**
- Collapsibles: height transition disabled, instant open/close
- Recharts: `isAnimationActive={false}` on all `<Area>` and `<Bar>` elements
- Hook decomp bar fill: jumps to final value instantly (no animate-fill)
- Skeleton shimmer: static gray, no `animate-pulse`
- AV transition: instant opacity change, no 200ms ease-out

**Perf tier degradation:**
- Low tier: Emotion arc renders (Recharts SVG is cheap), but `TapPopover` on dots disabled — `ReferenceDots` remain visible as static markers only
- Medium tier: full features
- Both tiers use `usePerfStore` from `@/lib/perf-tier`

## Anti-Patterns (Executor Must Avoid)

- **No glass treatment on inspector sheets / modals** — always solid opaque `bg-[#18191a]`
- **No red** — all failure/warning states use `--color-warning` (orange) or white/muted
- **No green** — no positive/negative color binary. Coral-intensity only.
- **No `VideoCard` from `/trending`** — build `SimilarVideoCardCompact` separately (wrong visual model, wrong deps)
- **No Radix Accordion for Verdict collapsibles** — use native `<details>` (lighter, same a11y)
- **No sentence-level sentiment classification on `result.reasoning`** — use `factors[]` score thresholds
- **No raw `cognitive_load` number surface** — always invert to Low/Med/High label
- **No `emotion_arc_peak_tapped` event on Low perf tier** — disable popover on dots, leave static ReferenceDots
- **No TikTok embed modal as inline mount** — always Radix Dialog portal at body level
- **No `board-constants.ts` hard-coded AV frame check** — all consumers use `getFrameAntiViralityState()` from `cross-group-state.ts`
- **No `overall_score` labeled as "percentile" when `is_calibrated === false`** — show `(score uncalibrated)` sub-text
- **No `backdrop-filter` via CSS class** — use `style={{ backdropFilter: 'blur(5px)' }}` (Lightning CSS strips it)
- **No `ResponsiveContainer height="100%"` on Emotion arc** — use explicit `height={140}` to avoid Pitfall 1
- **No `gap-1.5` (6px) in hook decomp bar stack** — use `gap-1` (4px); 6px is not on the 4-point scale
