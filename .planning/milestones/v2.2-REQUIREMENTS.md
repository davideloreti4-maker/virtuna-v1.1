# Requirements: Virtuna v2.2 Trending Page UI

**Defined:** 2026-02-05
**Core Value:** TikTok Creative Center-style trending feed built entirely with the v2.0 design system

## v2.2 Requirements

### Page Layout & Navigation

- [x] **PAGE-01**: Trending page accessible via sidebar nav item (TrendingUp icon)
- [x] **PAGE-02**: Page uses full-width layout within existing AppShell
- [x] **PAGE-03**: Page header with title ("Trending") using Typography Heading
- [x] **PAGE-04**: Route at `/trending` under the `(app)` route group

### Category Filtering

- [x] **FILT-01**: Tab bar with 3 categories: Breaking Out, Sustained Viral, Resurging
- [x] **FILT-02**: Tabs built with design system CategoryTabs/Tabs component
- [x] **FILT-03**: Active tab shows coral accent indicator
- [x] **FILT-04**: Tab switch filters the video grid without page reload

### Video Feed

- [x] **FEED-01**: Responsive grid of VideoCards (3 cols desktop, 2 tablet, 1 mobile)
- [x] **FEED-02**: Infinite scroll with Skeleton loading placeholders
- [x] **FEED-03**: Empty state when no videos match filter
- [x] **FEED-04**: Feed populated from mock data (20+ videos with realistic metadata)

### Video Card

- [x] **CARD-01**: VideoCard shows thumbnail image, creator name, view count, date
- [x] **CARD-02**: VideoCard shows category tag as GlassPill
- [x] **CARD-03**: VideoCard uses GlassCard with hover scale effect (HoverScale)
- [x] **CARD-04**: VideoCard shows trending velocity indicator (rising/peaking/declining arrow)
- [x] **CARD-05**: Click on VideoCard opens detail modal

### Video Detail Modal

- [x] **DETL-01**: Detail modal uses design system Dialog (lg or xl size)
- [x] **DETL-02**: Modal shows TikTok embed iframe (real embed from TikTok URL)
- [x] **DETL-03**: Modal shows full metadata: creator, views, likes, shares, date, hashtags
- [x] **DETL-04**: Modal has Analyze button (wired to existing Viral Predictor flow)
- [x] **DETL-05**: Modal has Bookmark button (toggles saved state, persisted in Zustand + localStorage)
- [x] **DETL-06**: Modal has Remix button showing "Coming Soon" tooltip/badge
- [x] **DETL-07**: Modal close via overlay click, escape key, or close button

### Bookmarks

- [x] **BMRK-01**: Bookmark state stored in Zustand with localStorage persistence
- [x] **BMRK-02**: Bookmarked videos show filled bookmark icon on their VideoCard
- [x] **BMRK-03**: Optional "Saved" filter tab to view bookmarked videos

### Mock Data

- [x] **MOCK-01**: Mock data file with 20+ trending videos across all 3 categories
- [x] **MOCK-02**: Each mock video has: id, title, thumbnail URL, creator, views, likes, shares, date, category, hashtags, tiktokUrl, velocity
- [x] **MOCK-03**: Realistic metadata values (view counts in thousands/millions, real-sounding creator names)

## Out of Scope

| Feature | Reason |
|---------|--------|
| Backend/API integration | UI-first milestone, backend comes later |
| Apify TikTok scraping | Deferred to backend milestone |
| AI classification | Deferred to backend milestone |
| Remix generation | Stub only, full system deferred |
| PDF export | Deferred to export milestone |
| Real-time trending data | Mock data sufficient for UI validation |
| Video editing tools | Different product category |
| Social features (comments/likes) | Not in scope for discovery page |
| Light mode | Dark-mode first, defer later |
