# Requirements: Virtuna -- Platform Refinement

**Defined:** 2026-02-16
**Core Value:** AI-powered content intelligence that tells TikTok creators whether their content will resonate -- and connects them to monetization opportunities.

## v1 Requirements

Requirements for this milestone. Each maps to roadmap phases.

### Sidebar & Navigation

- [ ] **NAV-01**: Sidebar design verified against brand bible (tokens, spacing, borders, typography)
- [ ] **NAV-02**: All sidebar buttons functional with correct navigation targets
- [ ] **NAV-03**: Duplicate "Content Intelligence" nav item resolved (removed or repurposed)
- [ ] **NAV-04**: Brand deals route shows proper page (Virtuna affiliate + "brand deals coming soon")
- [ ] **NAV-05**: Sidebar has visual harmony with overall platform aesthetic

### Landing Page

- [ ] **LAND-01**: Landing page has polished spacing, typography, and animations
- [ ] **LAND-02**: Hero section with compelling value proposition and clear CTAs
- [ ] **LAND-03**: Features section accurately represents current platform capabilities
- [ ] **LAND-04**: Stats section has meaningful, defensible numbers
- [ ] **LAND-05**: Social proof section refined (testimonials updated or restructured)
- [ ] **LAND-06**: FAQ section current and answers real user questions
- [ ] **LAND-07**: Footer polished and functional

### Trending Page

- [ ] **TREND-01**: Trending page has full redesigned UI layout
- [ ] **TREND-02**: Trending page shows meaningful content (mock data acceptable)
- [ ] **TREND-03**: Trending page properly integrated with navigation
- [ ] **TREND-04**: Trending page follows brand bible design patterns

### Settings

- [ ] **SET-01**: Profile section saves user data to Supabase
- [ ] **SET-02**: Password change functionality works end-to-end
- [ ] **SET-03**: Delete account functionality works with confirmation
- [ ] **SET-04**: Notification preferences save and persist
- [ ] **SET-05**: Team management has real functionality or clear coming-soon state
- [ ] **SET-06**: Settings UI matches brand bible design patterns

### Referrals & Brand Deals

- [ ] **REF-01**: Referrals page UI polished
- [ ] **REF-02**: Brand deals page shows Virtuna affiliate with "brand deals coming soon" section
- [ ] **REF-03**: Dead redirect replaced with proper page

### Auth & Onboarding

- [ ] **AUTH-01**: Signup page visually polished
- [ ] **AUTH-02**: Login page visually polished
- [ ] **AUTH-03**: Welcome/onboarding flow polished and smooth
- [ ] **AUTH-04**: Auth error states handled gracefully with clear messaging

## Future Requirements

Deferred beyond this milestone.

- AI-powered prediction engine integration (Backend Foundation milestone)
- Real TikTok data pipeline for trending page
- External brand deal listings from partner brands
- Light mode theme variant
- TikTok OAuth integration

Explicitly excluded from this milestone.

| Feature | Reason |
|---------|--------|

| Real trending data pipeline | Requires backend data infrastructure |
| External brand deal partnerships | Business development dependency |
| Light mode | Dark-mode first strategy unchanged |
| Mobile native app | Web-first |
| TikTok OAuth | Manual @handle sufficient |
