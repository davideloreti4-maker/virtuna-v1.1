# Juno — brand marks

> Provisional working marks (2026-07-03). Concept locked; final production cut (optical balance + custom wordmark) still open.

## Identity

**Numen Machines** (company) → **Juno** (creator app) → **Neura** (simulation model).
The mark belongs to **Juno**. See the naming architecture note for the full system.

## The mark — "The Resolve"

Three interconnected **audience** nodes converging to one **Read** node (terracotta).
It is a literal picture of what Juno does: an audience that connects *and resolves* into foresight. Category: intelligence / foresight — deliberately **not** a moon, orbit, or eye.

**Alternate — "The Signal":** four signals in a symmetric fan resolving to the Read. Simpler silhouette; strongest at micro sizes. Kept as a secondary option.

## Colour (from the app design system)

- Charcoal `#262624` — ground
- Cream `#ece7de` — mark (never pure white)
- Terracotta `#d97757` — **the Read node only.** One focal note per composition; removable (the mark reads in pure mono).

## Rules

- **Clear-space:** ≥ the diameter of the Read node (≈20 units) on all sides.
- **Min size:** use `juno-favicon.svg` below ~24px.
- **Wordmark:** Newsreader serif, "Juno". Lockups: mark left of wordmark (horizontal) or above (stacked).
- **Don't:** recolour more than the one Read node · add gradients/shadows/glow · distort the node/line ratio · fill the mark's background.
